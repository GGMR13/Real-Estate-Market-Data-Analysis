# Real-Estate-Market-Data-Analysis
Исследование объявлений о продаже квартир.

Проект: Анализ рынка недвижимости Санкт-Петербурга

Описание:
Проект по исследованию объявлений о продаже квартир с целью определения рыночной стоимости объектов и выявления факторов, влияющих на цену. В рамках работы был проведен исследовательский анализ данных (EDA), предобработка датасета и визуализация зависимостей.

Ключевые задачи:

Предобработка данных: Очистка данных, обработка пропусков и аномалий, приведение типов данных к корректному виду.
Исследовательский анализ (EDA): Изучение распределения цен, площадей и других характеристик квартир.
Анализ зависимостей: Определение влияния параметров (площадь, этаж, удаленность от центра) на итоговую стоимость.
Визуализация: Построение графиков и гистограмм для наглядного представления результатов.
Структура проекта:

real_estate_analysis.py — основной скрипт с кодом для анализа.
data/ — папка для хранения исходных данных (например, real_estate_data.csv).
notebooks/ — папка для Jupyter-ноутбуков (если используется).
reports/ — папка для сохранения отчетов и графиков.
